10-Year Stock Performance Analysis

An end-to-end data project analysing 10 years of equity performance (2014–2024) across AAPL, MSFT, MS, and IBM. Covers feature engineering, data pipeline (Yahoo Finance → SQLite → Tableau), and audience-specific dashboards. A custom Normalized Close metric enables fair, same-scale comparison across all four tickers.

🔍 Analytical Question

Traditional stock comparison using raw closing prices is misleading — stocks trade at very different price scales. This project engineers a Normalized Close feature that rebases all stocks to 1.0× at the start of 2014, enabling true like-for-like performance comparison across the full decade.

🔎 Results & Findings

Normalised returns rebased to 1.0× at start of 2014. Final values = end-of-period return multiples.

Stock Final Normalised Return Note
MSFT 15.03× ~15× its 2014 value
AAPL 15.00× ~15× its 2014 value
MS 5.77× ~5.8× its 2014 value
IBM 2.12× Clear underperformer
  • AAPL and MSFT are near-tied on raw terminal return (15.00× vs 15.03×) — neither is a clear winner on raw return alone.
  • AAPL’s edge is in consistency: smoother pre-2020 trajectory, volume dominance (~60B peak annual shares), and lower COVID crash drawdowns.
  • MSFT matched AAPL’s terminal return with higher volatility in specific periods, driven by Azure cloud re-rating post-2018.
  • IBM was the clear underperformer at 2.12× — barely above inflation over 10 years.
  • March 2020 COVID crash visible as a sharp dip; AAPL & MSFT recovered fastest, IBM notably slower.
  • AAPL volume declined from ~60B (2014) to ~5–10B (2024) — a sign of price appreciation, not declining interest.
